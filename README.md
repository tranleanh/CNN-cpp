# CNN-cpp
Convolutional Neural Network for MNIST Classification implemented in C++

OS: Windows 10

## USAGE:
1. Extract dataset:
```bashrc
mnist.rar
```
2. Navigate terminal to the repo, then run command: 
```bashrc
$ g++ cnn.cpp -o output
```
3. Execute output file: 
```bashrc
$ output.exe
```
